Requirements

  • A Windows 10 PC
  • The Steam version of Rocket League (Epic version might be supported but hasn't been tested)
  • [Bakkesmod](https://www.bakkesmod.com)
  • The RLGym plugin for Bakkesmod (It's installed automatically by pip)
  • Python >= 3.7

    Installation

    Alright future me who needs a reference on how to reinstall this stuff on a new computer, here it is:

    The most basic way to install rlgym is through the command line and by using pip install rlgym .

    If you are like me, you would be installing this in a virtual environment through anaconda. Make sure you have anaconda, then create a new virtual environment and through the anaconda command prompt (Hit windows then type "anaconda prompt" to get there) change to your current environment to your virtual environment through the command conda activate [name of your environment] If, for example, I had called my new environment 'tf', my command would be conda activate tf. You should see that your environment has changed (I think there's a difference somewhere on the left side of the command prompt). From there, install rlgym using pip install rlgym

    If you need to upgrade RLgym, use the command pip install --upgrade rlgym (while making sure you are in the correct virtual environment)

    Troubleshooting

    If for some reason everything seems like it should work but the game isn't changing when you run code, make sure that your injector didn't magically disappear (hey it happened to me once). You can check by going into the folder RLgym>plugins>RLGym.dll. If it isn't there, clone it from github [insert github link] and put it back.
